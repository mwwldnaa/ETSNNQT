Pada main.rs, terdapat program pub fn load_dataset(filename: &str) -> Result<Vec<(Array1<f64>, usize)>, Box<dyn Error>> {
    ...
}
program diatas berfungsi untuk membaca file CSV baris per baris, melewatkan header (if i == 0), memisahkan dan mengubah string menjadi angka (parse().unwrap_or(...)), dan menyimpan data dalam bentuk (fitur: Array1<f64>, label: usize). Tujuannya adalah untuk mengubah file CSV menjadi vektor numerik agar bisa digunakan neural network.

Program ini masih berupa neural network 1 layer, yakni prediksi acak yang belum dilakukan training.
pub struct NeuralNetwork {
    weights: Array2<f64>,
    bias: Array1<f64>,
}
Digunakan sigmuoid() sebagai fungsi aktivasi, weights dan bias diinisialisasi acak (rand::gen_range(...)), dan predict() untuk menentukan label dengan probabilitas tertinggi.

Program ini menggunakan dataset dengan format .csv yang berisi beberapa fitur diantaranya massa atau berat buah, widht, dan warna. Terdapat label angka 1 sampai 4 sesuai dengan jenais buah yakni apple, mandarin, orange, lemon. Kami menggunakan dataset yang bersumber dari github:
https://github.com/susanli2016/Machine-Learning-with-Python/blob/master/fruit_data_with_colors.txt

Digunakan 4 struktur utama, diantaranya:
1. main.rs >> sebagai program utama
2. model.rs >> berisi mengenai logika menggunakan metode neural network
3. dataset.rs >> sebagai penampung dari dataset.csv
4. dataset.csv >> berisi dataset buah yang digunakan.

Dari program yang telah dirancang diatas, digunakan beberapa variabel diantaranya:
1. inputs_size
2. output_size
3. nn
4. file
5. reader
6. dataset 
7. mass, width, height, color yang sama-sama menggunakan tipe data float 64
8. features
9. label

Beberapa fungsi yang digunakan:
main(): Fungsi utama program. Di dalamnya dipanggil fungsi load_dataset untuk memuat data, membuat objek neural network dengan NeuralNetwork::new(), dan melakukan prediksi menggunakan nn.predict().

load_dataset(...): Fungsi yang berasal dari file dataset.rs, digunakan untuk membaca data CSV dan mengubahnya menjadi vektor input dan label.

NeuralNetwork::new(...): Membuat model neural network dengan bobot dan bias yang diinisialisasi secara acak.

nn.predict(...): Fungsi yang digunakan untuk memprediksi hasil klasifikasi dari input fitur.

label_to_text(...): Mengubah label numerik hasil prediksi menjadi nama buah dalam bentuk string.

NeuralNetwork::new(input_size, output_size): Membuat instance neural network dengan ukuran input dan output yang ditentukan. Bobot dan bias diinisialisasi secara acak menggunakan crate rand.

predict(&self, input: &Array1<f64>) -> usize: Melakukan proses prediksi terhadap input fitur. Menggunakan operasi linear ditambah aktivasi sigmoid untuk menghasilkan prediksi label.

sigmoid(x: f64) -> f64: Fungsi aktivasi sigmoid yang digunakan untuk mengubah nilai linear ke dalam bentuk probabilitas (antara 0 dan 1).

label_to_text(label: usize) -> &'static str: Mengubah label numerik ke nama buah (misalnya: 1 menjadi "Apple").
